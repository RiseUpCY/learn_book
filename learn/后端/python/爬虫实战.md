## 常见问题

### 编码问题

```python
req = requests.get(url=url)
req.encoding            # 当前编码
req.apparent_encoding   # 页面的真实编码
req.encoding = 'GB2312' # 设为页面的真实编码，apparent_encoding
```

### 图片存储
